# Edge Computing Healthcare Project
☁️ Leveraging Machine Learning in Edge Computing for Healthcare with AWS IoT Greengrass, SageMaker and Lambda

Edge Computing is crucial for the future of healthcare as it enables real-time data processing at the source, facilitating immediate decision-making for critical patient care.

I designed a solution that utilises Edge Computing for low-latency, real-time health data processing, Machine Learning for predictive analytics on patient vitals, robust security measures for safeguarding sensitive patient information, and a blue-green CI/CD workflow to deploy updates efficiently.
